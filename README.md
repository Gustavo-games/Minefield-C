# Minefield-C

A simple Minesweeper clone built in C using SDL2 for rendering, input, text, and assets.

Compile with: `gcc main.c -lSDL2 -lGL -lSDL2_image -lSDL2_ttf -o main`

Run with: `./main` on the same file with both the wav and the images present
